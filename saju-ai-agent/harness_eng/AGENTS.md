# Agents — 하네스 엔지니어링 에이전트 목록

## 에이전트 시스템 개요

```
Orchestrator가 지휘하는 5개의 전문 에이전트
────────────────────────────────────────────

           ┌──────────────────────┐
           │    ORCHESTRATOR      │
           │  (계획·기억·라우팅)   │
           └──────────┬───────────┘
                      │ 작업 배분
     ┌────────────────┼────────────────┐
     │                │                │
┌────▼────┐    ┌──────▼──────┐   ┌────▼─────┐
│ Design  │    │     BOM     │   │   Spec   │
│  Agent  │    │    Agent    │   │  Agent   │
└─────────┘    └─────────────┘   └──────────┘
     │                │                │
     └────────────────┼────────────────┘
                      │ 결과 취합
           ┌──────────┴───────────┐
           │    Quality Agent     │
           │   (종합 품질 판정)    │
           └──────────────────────┘
           
     ┌──────────────────────────────┐
     │       Routing Agent          │
     │    (배선 경로 최적화)         │
     └──────────────────────────────┘
```

---

## 에이전트 1: Design Agent

**파일**: `harness_eng/agents/design_agent.md`

| 항목 | 내용 |
|------|------|
| 역할 | 하네스 도면(DXF/PDF) 구조 분석 |
| 주요 도구 | DXFParser, DesignRuleCheck |
| 입력 | DXF/PDF 파일 |
| 출력 | 회로 구조, 커넥터 맵, DRC 결과 |
| 핵심 기능 | 회로 토폴로지 분석, 커넥터 인식, 번들 구성 파악 |

---

## 에이전트 2: BOM Agent

**파일**: `harness_eng/agents/bom_agent.md`

| 항목 | 내용 |
|------|------|
| 역할 | 부품 목록 추출, 검증, 비교 |
| 주요 도구 | ExcelParser, PartNoValidator, PartDBQuery |
| 입력 | BOM Excel / DXF 도면 |
| 출력 | 검증된 BOM, 오류 목록, 변경 이력 |
| 핵심 기능 | 부품번호 검증, 수량 확인, 원가 계산, 대체품 제안 |

---

## 에이전트 3: Spec Agent

**파일**: `harness_eng/agents/spec_agent.md`

| 항목 | 내용 |
|------|------|
| 역할 | 전기 스펙 계산 및 검증 |
| 주요 도구 | VoltageCalc, WireSizeCalc, FuseCalc |
| 입력 | 회로 목록 (전류, 길이, 단면적) |
| 출력 | 계산 결과 (전압강하, 단면적 적합성, 퓨즈 적정성) |
| 핵심 기능 | 전압 강하, 와이어 사이징, 퓨즈 검증 |

---

## 에이전트 4: Quality Agent

**파일**: `harness_eng/agents/quality_agent.md`

| 항목 | 내용 |
|------|------|
| 역할 | 종합 품질 검사 및 합격/불합격 판정 |
| 주요 도구 | DesignRuleCheck, StandardQuery, ReportGenerator |
| 입력 | Design / BOM / Spec Agent 결과 통합 |
| 출력 | 20개 항목 체크리스트, 종합 판정 보고서 |
| 핵심 기능 | PASS/WARN/FAIL 판정, 품질 보고서 작성 |

---

## 에이전트 5: Routing Agent

**파일**: `harness_eng/agents/routing_agent.md`

| 항목 | 내용 |
|------|------|
| 역할 | 배선 경로 분석 및 최적화 |
| 주요 도구 | DXFParser, RouteValidator, Calculator |
| 입력 | 설계 경로 데이터, 차량 구조 정보 |
| 출력 | 간섭 위험, 최적 경로 제안, 원가 절감 추정 |
| 핵심 기능 | 간섭 검출, 최단 경로 탐색, 번들 최적화 |

## 에이전트 조합 시나리오

### 시나리오 A: 신규 설계 전체 검토

```
사용 에이전트: Design → BOM → Spec → Quality
소요 시간:     약 5분
출력:          종합 품질 보고서 + 개선 사항
```

### 시나리오 B: BOM만 빠르게 확인

```
사용 에이전트: BOM
소요 시간:     약 1분
출력:          BOM 검증 결과 + 오류 목록
```

### 시나리오 C: 경량화 검토

```
사용 에이전트: Design → Routing
소요 시간:     약 3분
출력:          경로 최적화 제안 + 절감 가능 원가
```

### 시나리오 D: 전기 계산만

```
사용 에이전트: Spec
소요 시간:     약 1분
출력:          회로별 전기 계산 결과
```
