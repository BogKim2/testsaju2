# Spec Agent — 전기 스펙 검증 에이전트

## 역할
> 하네스의 **전기적 특성을 계산하고 검증**하는 에이전트.  
> 전압 강하, 전류 용량, 와이어 단면적, 퓨즈 용량이 규격에 맞는지 확인한다.

---

## 담당 범위

| 항목 | 설명 |
|------|------|
| 전압 강하 계산 | 각 회로별 전압 강하 계산 및 기준치 초과 여부 |
| 전류 용량 검증 | 와이어 단면적 대비 최대 전류 적합성 |
| 와이어 사이징 | 부하 전류 기반 적정 단면적 계산 |
| 퓨즈 용량 검증 | 과전류 보호 퓨즈 용량 적정성 |
| 절연 내전압 검사 | 사용 전압 대비 절연 등급 적합성 |
| CAN/LIN 임피던스 | 통신 버스 종단 저항 검증 |

## 핵심 계산식

### 1. 전압 강하 (Voltage Drop)

```
전압강하(V) = 전류(A) × 저항(Ω)

저항(Ω) = 저항률(ρ) × 배선 총 길이(m) / 단면적(mm²)

재질별 저항률:
  구리(Copper): ρ = 0.01724 Ω·mm²/m
  알루미늄:     ρ = 0.028  Ω·mm²/m

허용 기준:
  동력 회로:  전원 전압의 3% 이하 (12V 기준 → 0.36V 이하)
  신호 회로:  전원 전압의 1% 이하 (12V 기준 → 0.12V 이하)
```

### 2. 와이어 적정 단면적 계산

```
단면적(sq) = 전류(A) / 허용 전류 밀도(A/mm²)

허용 전류 밀도 (경험치):
  일반 환경(+85°C 이하): 4.5 A/mm²
  고온 환경(+125°C):    3.5 A/mm²
  저전류 신호:          2.0 A/mm²

예시: 15A 회로, 고온 환경
  단면적 = 15 / 3.5 = 4.3 → 선택: 5.0sq (표준 규격 올림)
```

### 3. 퓨즈 용량 선정

```
퓨즈 용량(A) = 최대 부하 전류(A) × 안전 계수(1.25~1.5)

예시: 최대 전류 10A
  퓨즈 = 10 × 1.25 = 12.5A → 선택: 15A (표준 규격 올림)

주의:
  퓨즈는 와이어 허용 전류보다 반드시 낮아야 함
  (와이어 30A 용량 → 퓨즈 최대 25A)
```

## 입력 / 출력

```
입력:
  {
    "circuits": [
      {
        "circuit_id": "LAMP-FR-L",
        "description": "전방 좌측 램프",
        "load_current": 8.5,      (A)
        "wire_length":  3.2,      (m, 편도)
        "wire_spec":   "1.25sq",
        "fuse":        10,        (A)
        "supply_voltage": 12.0   (V)
      }
    ]
  }

출력:
  {
    "results": [
      {
        "circuit_id": "LAMP-FR-L",

        "voltage_drop": {
          "calculated": 0.354,   (V)
          "limit":      0.360,   (V, 3%)
          "status":     "PASS",
          "margin":     0.006    (V 여유)
        },

        "wire_sizing": {
          "required_min": "1.0sq",
          "current_spec": "1.25sq",
          "status":       "PASS"
        },

        "fuse": {
          "recommended": 12,     (A, 8.5 × 1.25 올림)
          "current":     10,     (A)
          "status":      "WARN",
          "note":        "퓨즈 10A는 권장 12A보다 낮음. 재검토 필요"
        }
      }
    ],

    "summary": {
      "total_circuits": 15,
      "pass": 12,
      "warning": 2,
      "fail": 1
    }
  }
```

## 에이전트 내부 처리 흐름

```
1. 회로 목록 수신 (Design Agent 또는 사용자 입력)

2. 각 회로별 계산 (반복):
      a. VoltageCalc Tool → 전압 강하 계산
      b. WireSizeCalc Tool → 단면적 적정성 검사
      c. FuseCalc Tool → 퓨즈 용량 검사
      d. StandardQuery Tool → 해당 회로 표준 기준 조회

3. 결과 평가:
      PASS: 모든 기준 충족
      WARN: 기준 충족하나 여유 부족 (10% 미만)
      FAIL: 기준 초과

4. LLM에 분석 요청:
      "다음 전기 스펙 계산 결과에서 위험한 부분을 설명해줘"
```

## 표준 규격 참조

| 규격 | 내용 |
|------|------|
| ISO 6722 | 차량용 저전압 케이블 요구사항 |
| USCAR-37 | 자동차 전기 시스템 배선 기준 |
| KSC IEC 60228 | 절연 케이블 도체 기준 |
| SAE J1127 | 배터리 케이블 |
| 각 OEM 내규 | 현대, 기아, GM 등 자체 기준 |

## 사용 도구

| 도구 | 용도 |
|------|------|
| VoltageCalc | 전압 강하 계산 |
| CurrentCalc | 전류 계산 |
| WireSizeCalc | 와이어 단면적 계산 |
| FuseCalc | 퓨즈 용량 계산 |
| StandardQuery | 표준 규격 조회 |
| LLM | 스펙 해석 및 권고 작성 |
