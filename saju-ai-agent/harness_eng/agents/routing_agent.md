# Routing Agent — 배선 경로 최적화 에이전트

## 역할
> 하네스의 **배선 경로(Route)를 분석하고 최적화**하는 에이전트.  
> 최단 경로, 간섭 회피, 번들 최적화를 통해 원가와 무게를 줄인다.

---

## 담당 범위

| 항목 | 설명 |
|------|------|
| 경로 분석 | 현재 배선 경로의 총 길이 및 구조 파악 |
| 간섭 검출 | 다른 부품(엔진, 배기, 샤시)과의 간섭 확인 |
| 경로 최적화 | 더 짧고 안전한 대안 경로 제안 |
| 번들 최적화 | 동일 경로 와이어를 번들로 묶어 정리 |
| 클램프 포인트 | 클램프 위치 및 간격 최적화 |
| 무게/원가 절감 | 경로 최적화로 와이어 길이 단축 → 원가 절감 |

---

## 경로 최적화 핵심 원칙

```
원칙 1: 최단 경로 우선
  → 불필요한 우회 배제
  → 와이어 길이 최소화

원칙 2: 간섭 회피 필수
  → 고온 부위 (배기관 200mm 이상 이격)
  → 회전 부품 (팬벨트 50mm 이상 이격)
  → 날카로운 모서리 회피

원칙 3: 번들링 최대화
  → 동일 방향 와이어는 묶어서 관리
  → 번들 크기 최대 지름 25mm (클램프 기준)

원칙 4: 정비성 확보
  → 탈착이 쉬운 구조
  → 커넥터 접근 공간 확보
```

## 간섭 검출 기준

### 온도별 최소 이격 거리

| 발열 부위 | 최소 이격 |
|---------|---------|
| 배기관 (400~600°C) | 200mm |
| 촉매 변환기 | 250mm |
| 터보차저 | 300mm |
| 브레이크 디스크 | 100mm |
| 엔진 블록 | 50mm |

### 기계적 간섭 기준

| 부위 | 최소 이격 |
|------|---------|
| 팬벨트/구동 벨트 | 50mm |
| 조향 로드 | 30mm |
| 차축/드라이브 샤프트 | 50mm |
| 날카로운 금속 모서리 | 그로멧 또는 보호재 필수 |

## 번들 최적화 로직

```
번들링 기준:
  1. 동일 출발지 → 동일 목적지 경로
  2. 전력 회로와 신호 회로 분리 원칙
     (EMI 간섭 방지)
  3. 최대 번들 직경 25mm 준수

번들 분류:
  파워 번들:  배터리, IGN, 고전류 회로
  신호 번들:  센서, 통신, 저전류 회로
  접지 번들:  모든 GND 회로

번들 직경 계산:
  d_total = 1.15 × √(Σ d_wire²)
  (15% 패킹 여유율 적용)
```

## 에이전트 내부 처리 흐름

```
1. Design Agent 결과에서 경로 데이터 추출
      → 와이어 시작점, 끝점, 경유 포인트

2. 현재 경로 총 길이 계산
      → 모든 와이어 합산

3. 간섭 검출
      → 차량 구조 데이터와 경로 좌표 비교
      → 허용 이격 기준 위반 탐지

4. 최적 경로 탐색 (알고리즘)
      → 간섭 없는 공간에서 최단 경로 계산
      → A* 알고리즘 또는 Dijkstra 적용

5. 번들 최적화
      → 동일 방향 와이어 그룹화
      → 번들 직경 계산

6. 클램프 위치 최적화
      → 300mm 간격 기준 클램프 포인트 설정

7. LLM에 최적화 제안 작성 요청
```

## 품질 기준

| 지표 | 기준 |
|------|------|
| 경로 길이 계산 정확도 | ±5mm 이내 |
| 간섭 검출률 | 95% 이상 |
| 최적화 제안 소요 시간 | 30초 이내 |
| 평균 원가 절감 기여 | 총 와이어 길이 5~15% 단축 |
